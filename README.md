## 서론

이 프로젝트는 DDD를 기반으로 MSA를 구성한 프로젝트입니다. Spring Boot를 이용하여 Back-end의 Template를 구성하였으며, AWS EKS를 이용해 각 마이크로 서비스들의 Service,
Deployment를 생성하여 서비스를 배포할 수 있도록 구성하였습니다.
